Örnek #04: Paragrafta Alt Satıra Geçmek, Metinler Arasına Çizgi Çekmek

Buradaki metin baya bir uzun olarak yazıldı. Normalde paragrafta
ENTER ile alt satıra da geçsek HTML'de bunun bir anlamı yoktur.
Alt satıra heçmek için "< br >" kullanılır. Düz çizgi ile metin bloklarını
ayırmak içinse "< hr >" kullanılır. 

    Ayrıca burada yazılan yazıda olduğu gibi ENTER tuşunun
    ( Windows'ta: CRLF (\r\n), MAC'te: CR (\r) ) işlevinin
    olması için "< pre >" kullanılır. (preformatted text)

Örnek #05: "Inline" Stiller

Bu yazı, başlık 3 olarak yazıldı. Arka planı mavi renk. Temaya göre yazı rengi değişiyor.

Bu yazı, başlık 3 olarak mavi renk üzerine sarı olarak yazıldı ama CSS dosyasındaki default
yazı rengi (* {...} kısmı ) yüzünden istediğimiz gibi olmuyor!

Bu yazı, başlık 3 olarak yeşil renk üzerine siyah olarak yazıldı ama CSS dosyasındaki default
yazı rengi (* {...} kısmı ) yüzünden istediğimiz gibi olmuyor! Renk bilgisi yazısı normalin 1,5 katı olarak seçildi.

Bu paragrafın arka planı domates rengi ve altı çizgili olarak yazıldı.

Bu yazının "burası" kırmızı üzerine beyaz olarak yazıldı. Renkli kısım temayla değişmiyor. ("span" kullanıldı) Yazı tipi "Segoe UI" ailesi seçildi. Metin ortalandı ama işlev görmedi.

Örnek #07: HTML Renkleri

  • VS Code Studio'da mesela "color:" ifadesi otomatik olarak tanınır ve renk için öneri kutusu, renk paleti ile beraber açılır. Ön tanımlı renk istenmezse paletten renk seçilebilir.
  • Öntanımlı bazı renkler: Tomato, Orange, DodgerBlue, MediumSeaGreen, Gray, SlateBlue, Violet, LightGray
  • Üç farklı şekilde renk girilebilir: RGB, HEX ve HSL olarak. Opaklık verilmesi istenirse RGBA veya HSLA tercih edilir. (A: Alpha) 
                    * 16.777.216 (16M) farklı renk seçeneği vardır. ( 2563 )
                * HEX olarak giriş yapılacaksa FF(RED)  63(GREEN) 47(BLUE) tonları olarak düşünülmelidir.
                * RGB'de değerler 0-255, HEX'te ise 00-FF arasında olabilir.
                * Hue Saturation Lightness, HSL formatını ifade eder. İlk değer 0-359 arasında olabilir. Diğerleri yüzde olarak 0-100 arasındadır.

    TEST METNİ -- rgb(255,99,71)

    TEST METNİ -- #FF6347

    TEST METNİ -- hsl(9,100%,64%)

    TEST METNİ -- rgba(255,99,71,0.75)

    TEST METNİ -- hsla(9,100%,64%,0.75)
